本專案是一套結合 即時交通資料（TDX） 與 時空圖神經網路（ST-GNN） 的智慧交通分析系統，透過地理資訊視覺化技術，呈現台北市道路即時壅塞狀態與交通流動情形，並提供 AI 輔助分析與預測能力。

系統以 Mapbox 為地圖核心，將不同路段的即時速度與壅塞程度進行視覺化標記（綠 / 黃 / 紅），讓使用者能快速掌握城市交通狀況。

核心功能 即時交通監控
串接 TDX 即時路段速度資料
每分鐘更新道路狀態
顯示不同壅塞等級（順暢 / 壅塞 / 重大壅塞）
AI 交通預測（ST-GNN）
使用時空圖神經網路預測未來路段速度
支援短期交通趨勢預測
地理視覺化系統
基於 Mapbox 的互動式地圖
動態路網渲染（道路顏色代表壅塞程度）
支援區域放大與路段觀察
GeoAI Assistant（可選）
AI 交通助理介面
提供即時路況摘要與建議
可查詢特定路段或區域
